# ระบบยืม-คืนอุปกรณ์ ภาควิชารังสีเทคนิค

## วิธีติดตั้งและรัน

### 1. ติดตั้ง Python packages
```bash
pip install -r requirements.txt
```

### 2. รันแอป
```bash
streamlit run app.py
```

จากนั้นเปิดเบราว์เซอร์ไปที่ `http://localhost:8501`

---

## Deploy บน Streamlit Cloud (ฟรี)

1. สร้าง GitHub Repository ใหม่
2. อัปโหลดไฟล์ `app.py` และ `requirements.txt`
3. ไปที่ [share.streamlit.io](https://share.streamlit.io)
4. เชื่อมต่อ GitHub และเลือก repo
5. กด **Deploy** — ได้ URL สาธารณะฟรี!
